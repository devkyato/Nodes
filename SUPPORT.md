# Support

## Before opening an issue

- Refresh the page and retry in a current Chrome, Edge, Firefox, or Safari release.
- Export the project as JSON before clearing browser data or testing destructive steps.
- Try the same action with a small new diagram to separate project-data problems from editor behavior.
- Search existing issues for the same browser and interaction.

## Getting help

Use the repository's bug report form for reproducible problems and the feature request form for focused improvements. Include the browser version, exact steps, and a sanitized project JSON when it helps reproduce the behavior.

Do not attach confidential diagram content. Report security problems privately by following [SECURITY.md](SECURITY.md).

## Common recovery steps

- Use Undo after an accidental clear, delete, move, or property change.
- Use Load to restore the last project saved in this browser.
- Import a previously downloaded JSON file to recover a portable project copy.
- If a project file cannot be imported, verify that it is valid JSON and contains `nodes` and `edges` arrays.
